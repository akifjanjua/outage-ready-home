# Outage Ready Home

Outage Ready Home is a niche content website built around practical power outage preparation for everyday homes, apartments, renters, remote workers, and families.

The site includes outage planning guides, interactive calculators, SEO-friendly page structure, internal linking, sitemap, robots file, Open Graph image support, and clean responsive design.

## Live Website

https://outagereadyhome.vercel.app/

## Local Development

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open the local website:

```txt
http://localhost:3000
```

Build for production:

```bash
npm run build
```

## Deployment

The project is deployed on Vercel and connected to GitHub. Any pushed changes to the main branch can trigger a new deployment.
